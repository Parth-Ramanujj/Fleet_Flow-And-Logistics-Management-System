---
description: Steps to upgrade FleetFlow from MVP to production with React + Supabase
---

1. Install Supabase and supporting dependencies
2. Set up Supabase project and configure environment variables
3. Build the full database schema in Supabase
4. Implement the design system and global styles upgrade
5. Build reusable shared UI components library
6. Build AuthContext and Supabase auth integration (Login + RBAC)
7. Build ProtectedRoute and role-based layout
8. Wire Vehicles page to Supabase (CRUD + real-time)
9. Wire Drivers page to Supabase (CRUD + real-time)
10. Wire Trip Dispatcher to Supabase (CRUD + validation logic)
11. Wire Maintenance Logs to Supabase
12. Wire Fuel/Expense Logs to Supabase
13. Wire Dashboard KPIs to real Supabase aggregate queries
14. Build Analytics page with real charts (recharts)
15. Final polish: toasts, error boundaries, loading skeletons, responsive mobile sidebar
