FleetFlow is a modern web application that simplifies the complex logistics behind managing a fleet of vehicles and drivers. It offers a centralized dashboard that tracks live trips, vehicle maintenance schedules, fuel logs, and driver safety scores.

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Parth-Ramanujj/Fleet_Flow-And-Logistics-Management-System.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Create a `.env` file in the root directory and connect your Supabase project:
   ```env
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
